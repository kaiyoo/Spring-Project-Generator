== 코딩 규칙 ==

01. 다함께 보는 코드이기 때문에 들여쓰기 제대로 하자.(4 space)

02. 쿼리문 작성시 조건절에 '' 들어갈 값 외에는 대문자로 기술한다.

03. 클래스, 함수, 조건문, 반복문에는 반드시 주석 처리.

04. 코드 작성시 코드 최상단에 등록자, 등록일을 기재.

05. 코드 수정시 코드 최상단에 수정자, 수정일을 기재.

06. SVN에 코드 등록시 sync 확인(ctrl + alt + s)
   -> 충돌 여부 확인후 이상없을경우에만 commit 수행

07. commit 시 절대로 override and commit 금지 !!!!

08. 타인 코드 수정시 담당자에게 미리 말하고 수정하자.

09. DB 데이터 변경시(insert, update, delete) 반드시 commit 수행.
